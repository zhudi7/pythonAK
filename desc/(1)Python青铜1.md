## Mac下pip的安装

打开终端运行以下命令

```python
curl https://bootstrap.pypa.io/get-pip.py | python3
```

安装后查看版本

```python
pip --version
```

查看相应的包

```python
pip3 list
```

安装和更新pip

```python
pip install --upgrade pip
```
